## Packages
@dnd-kit/core | Core drag and drop functionality for the Kanban board
@dnd-kit/utilities | Utilities for Dnd kit
@dnd-kit/sortable | Sortable logic for drag and drop
date-fns | Date formatting for countdowns and standups
recharts | Beautiful charts for project dashboard
next-themes | Dark/Light mode theme switching

## Notes
- Wouter handles routing. Wouter `<Link>` elements render directly as `<a>`, no nesting.
- TanStack Query handles API fetching.
- Drag and Drop uses dnd-kit with basic useDroppable/useDraggable for Kanban columns.
- The UI features a hackathon "Cyber/Neon" aesthetic with Rajdhani for display text and JetBrains Mono for numbers.
